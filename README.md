# PAT 2 - Sub-task 1: Morse Code Research

## 1. What is Morse Code?
Morse code is a simple way of sending text messages using just two signs: dots (short signals) and dashes (long signals). Instead of typing letters normally, you change each letter into a pattern of these dots and dashes. People use it to send messages over long distances using things like flashing lights, sound beeps, or radio signals.

## 2. History of Morse Code
Morse code was invented a long time ago, back in the 1830s. It was made by two men named Samuel Morse and Alfred Vail because they were trying to make the electric telegraph work. 

Before this, sending messages far away was really hard and took a long time. They made Morse code so messages could travel quickly through copper wires using electrical clicks. To make typing faster, Alfred Vail looked at which letters people use the most in English. For example, the letter 'E' is used all the time, so he gave it the shortest code: just one single dot.

## 3. How the System Works
The system changes regular English letters and numbers into a timing pattern of dots and dashes. 

### The Rules:
* **Dot (`.`):** This is the short sound or light click. It is the basic unit of time.
* **Dash (`-`):** This is the long sound. A dash is exactly as long as 3 dots put together.
* **Inside a letter:** The space between the dots and dashes inside the exact same letter is equal to 1 dot.
* **Between letters:** When you finish a letter and move to the next one, you wait for 3 dots of silence.
* **Between words:** When you finish a whole word, you wait for 7 dots of silence before starting the next word.
